# AUBIElang
The Aubie programming language

Syntax rules:
  statements begin with a keyword followed by arguments
  ex:
    print "Hello World!"
    sum 1 2 3 4
   
  variable definitions are of the format: name type value
    valid variable types are: num, string
    value can be a function
    ex:
      def a num 23.2
      def b string "AUBIE"
      def c mult 3 5
